# Dotfiles

My personalized scripts and aliases.

## Installation

  1. Clone: `git clone https://github.com/holman/dotfiles.git`
  2. Add an environment variable for DOTFILES that points to the cloned project.
  3. source `${DOTFILES}/aliases`
